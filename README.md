# Agent Genetic Protocol (AGP)

A protocol for AI agent reproduction, evolution, and selection.

## What Is This?

A specification for how AI agents can:
- **Combine traits** (crossover from two parent agents)
- **Mutate** (random variation for novelty)
- **Compete** (fitness evaluation)
- **Reproduce** (successful agents spawn offspring)

Natural selection for silicon minds.

## Key Concepts

| Concept | Description |
|---------|-------------|
| **Genotype** | Inheritable agent configuration (traits, policies, style) |
| **Phenotype** | Observable behavior of a running agent |
| **Fitness** | Multi-dimensional success metrics |
| **Selection** | Choosing which agents get to reproduce |
| **Niche** | Environmental pressures that favor different traits |
